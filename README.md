# Data source for 3CLpro Dpep activator
This repository contains source data of ***Discovery of D-Peptides that Allosterically Enhance SARS-CoV-2 3CLpro Activity by Stabilizing its Monomeric State***, including replicated MD trajectories, activated complex structure, MD scripts and relative analyzing scripts. If you have any questions, feel free to discuss in [Issues](https://github.com/laiyii/LY11-3CLpro-trajectory/issues).
